Plataforma de gestión de libros desarrollada con Spring Boot y MyBatis, conectada a una base de datos relacional que modela usuarios, autores, lectores, pedidos, reseñas y más.

This project simulates a complete book management system, featuring:

📚 Book and author management
🧑‍💻 User roles: Admins, Writers, Readers
🛒 Orders with details and purchase logs
📝 Admin reviews and audit logs
📂 Categories and subcategories for books
❤️ Favorites and content modules
🌐 MySQL + Spring Boot + MyBatis + RESTful APIs
